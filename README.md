# linux-vfat-journal
MYY601 lab on journaling support in the Linux VFAT file system

# MYY601 - Εργαστήριο 2
## Υλοποίηση αρχείου καταγραφής στο σύστημα αρχείων VFAT του Linux

## Περιγραφή
Η παρούσα εργασία αφορά την υλοποίηση μηχανισμού καταγραφής (journal) στο σύστημα αρχείων VFAT του Linux.

Οι αλλαγές πραγματοποιήθηκαν πάνω σε συγκεκριμένα αρχεία του δοσμένου πηγαίου κώδικα της LKL/VFAT υλοποίησης, με στόχο την καταγραφή μεταβολών σε βασικές δομές του συστήματος αρχείων πριν την ενημέρωσή τους.

## Περιεχόμενα Repository
Το repository περιλαμβάνει μόνο τα αρχεία που τροποποιήθηκαν για τις ανάγκες της εργασίας και όχι ολόκληρο τον αρχικό κώδικα.

## Σημείωση για τον αρχικό κώδικα
Ο αρχικός κώδικας δόθηκε ως εκπαιδευτικό υλικό στο πλαίσιο του μαθήματος και δεν αναδημοσιεύεται αυτούσιος εδώ.
Περιλαμβάνονται μόνο οι δικές μου τροποποιήσεις πάνω στον δοσμένο κώδικα.
